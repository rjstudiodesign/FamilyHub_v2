# Family Management UI Guide

## Page Layout

The Family Management page consists of three main sections arranged vertically:

1. **Family Selector** - Top section for switching between families
2. **Family Details** - Middle section with statistics and settings
3. **Members List** - Bottom section with member and child profile management

## Color Scheme

### Status Colors
- **Primary**: Accent glow (blue/cyan gradient)
- **Success**: Green (#10b981)
- **Warning**: Orange (#f59e0b)
- **Error**: Red (#ef4444)
- **Info**: Blue (#3b82f6)

### Component Colors
- **Admin Badge**: Accent glow with 20% opacity background
- **Danger Actions**: Red with 20% opacity background
- **Child Profile Icon**: Pink to purple gradient
- **Settings Icons**: Various colors matching function

## Responsive Behavior

### Desktop (>1024px)
- Full width layout with max-width constraint
- Cards displayed in grid
- All buttons visible with text

### Tablet (768px - 1024px)
- Single column layout
- Buttons may be icon-only
- Modals remain centered

### Mobile (<768px)
- Stacked layout
- Full-width buttons
- Simplified navigation
- Touch-optimized spacing

## Accessibility

- All buttons have aria-labels
- Proper heading hierarchy
- Keyboard navigation support
- Screen reader friendly
- High contrast mode compatible

## Animations

- Fade-in on page load
- Smooth transitions on hover
- Modal slide-in effects
- Loading spinners
- Toast notifications

## Icons

Using Lucide icons:
- `users`: Family/members
- `user-plus`: Add member
- `baby`: Child profile
- `shield`: Admin/security
- `settings`: Settings
- `edit`: Edit action
- `trash-2`: Delete action
- `alert-triangle`: Warning
- `plus`: Create/add
- `log-out`: Leave

## Notifications

Toast notifications appear at top-right:
- Success: Green with checkmark
- Error: Red with X
- Warning: Orange with alert triangle
- Info: Blue with info icon

Auto-dismiss after 3 seconds.
