# FamilyHub

[![Netlify Status](https://api.netlify.com/api/v1/badges/3d924148-9f43-4cea-b3ad-9dec49aa4ba0/deploy-status)](https://app.netlify.com/projects/familyhubrjstudio17/deploys)

Willkommen bei FamilyHub! Eine private Social-Media-Plattform für Familien.

## Entwicklung

1.  **Abhängigkeiten installieren:**
    ```bash
    npm install
    ```
2.  **Entwicklungsserver starten:**
    ```bash
    npm run dev
    ```
3.  **Firebase Emulators starten (in einem separaten Terminal):**
    ```bash
    firebase emulators:start
    ```
